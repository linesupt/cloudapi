## 项目介绍

### 项目架构
* JDK 1.8
* Springboot 1.5.6.RELEASE
* Mybatis SpringBoot starter 1.3.0
* thymeleaf 前端模板引擎
* alibaba druid 连接池
* pagehelper SpringBoot starter 1.1.2
* mybatis generator 自动生成代码插件 1.3.2


### 主要功能
* 集成了Mybatis
* mpxj读取.mpp格式的Project文件（2018-01-09更新）
* mpxj导出.xml格式的Project导入向导文件（2018-01-15更新）
